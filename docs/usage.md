# 使用指南

## 准备查询数据

每个需要绘制的 Grafana 查询都应返回一个 DataFrame，其中包含：

- 纬度字段，默认匹配 `lat`
- 经度字段，默认匹配 `(lon|lng)`
- 可选的悬浮提示字段

经纬度匹配不区分大小写，可填写准确字段名或正则表达式。无效正则会按普通文本处理。
每个查询使用自己的 `refId`（如 `A`、`B`）选择绘制方式。

轨迹按照查询返回的行顺序连接有效坐标，至少需要两个有效点。标点为每个有效坐标绘制
一个圆形标记。空值、非数字以及超出纬度 `[-90, 90]` 或经度 `[-180, 180]` 的坐标会
被忽略。

## 基本配置

1. 新建或编辑 Grafana 面板，选择 **Ageomap**。
2. 按[安装与配置](setup.md#配置高德凭据)填写高德 Key 和 `securityJsCode`。
3. 在“数据映射”中选择源数据坐标系。
4. 设置经纬度字段匹配表达式。
5. 在“按 refId 配置 (JSON)”中指定每个查询是轨迹、标点还是不绘制。

默认源坐标系为 WGS-84，面板会调用高德官方 `AMap.convertFrom` API 转换为 GCJ-02。
数据已经是 GCJ-02 时，请选择 GCJ-02，避免重复转换。转换按高德单次上限分成最多
40 对坐标的批次顺序执行；调用失败或 15 秒超时会显示在面板上。

## 按查询配置

配置值是以查询 `refId` 为键的 JSON 对象：

```json
{
  "A": {
    "kind": "route",
    "color": "#1F60C4",
    "lineWidth": 4,
    "opacity": 0.6,
    "zIndex": 10
  },
  "B": {
    "kind": "marker",
    "color": "#C4162A",
    "radius": 5,
    "opacity": 0.8,
    "zIndex": 100,
    "labelFields": "name,value",
    "labelSuffix": " kWh"
  },
  "C": {
    "kind": "none"
  }
}
```

| 属性          | 适用类型   | 说明                               |
| ------------- | ---------- | ---------------------------------- |
| `kind`        | 全部       | `route`、`marker` 或 `none`        |
| `color`       | 轨迹、标点 | CSS 颜色；未设置时使用全局默认值   |
| `opacity`     | 轨迹、标点 | 透明度；未设置时使用全局默认值     |
| `zIndex`      | 轨迹、标点 | 高德覆盖物层级                     |
| `lineWidth`   | 轨迹       | 折线宽度                           |
| `radius`      | 标点       | 圆形标点半径                       |
| `labelFields` | 标点       | 逗号分隔的准确字段名，用于悬浮提示 |
| `labelSuffix` | 标点       | 附加到数值提示后的文本             |

`labelFields` 按配置顺序用 `·` 连接非空值。数值会四舍五入到整数，再附加
`labelSuffix`；字符串保持原样。提示内容以纯文本插入，不会作为 HTML 执行。

未配置 `kind` 时，第一个 DataFrame 默认为 `route`，后续 DataFrame 默认为
`marker`。默认 JSON `{"A":{"kind":"route"}}` 明确把查询 A 配置为轨迹。

配置不是有效 JSON、顶层不是对象、属性类型错误或 `kind` 无效时，面板会显示错误并
停止绘制。

## 全局面板选项

### 地图样式

- **跟随 Grafana**：浅色和深色主题分别选择地图样式。
- **固定**：始终使用指定地图样式。

可用样式为 `normal`、`dark`、`light`、`whitesmoke`、`fresh`、`grey`、
`graffiti`、`macaron`、`blue`、`darkblue` 和 `wine`。

### 视图

- **自动缩放到数据范围**：有覆盖物时调用高德地图适应视野。
- **默认缩放级别**：默认值为 11；自动缩放关闭或尚无数据时使用。
- **显示工具条**：加载高德 `ToolBar` 控件。
- **显示比例尺**：加载高德 `Scale` 控件。

### 默认样式

“轨迹样式”设置轨迹颜色、宽度和透明度；“标点样式”设置标点颜色、半径和透明度。
按 `refId` 配置中的同名属性会覆盖全局默认值。

## 常见问题

### 查询有数据但没有图形

检查查询 `refId`、`kind`、经纬度字段匹配、坐标范围和坐标系。轨迹只有一个有效点时
不会绘制。

### 悬浮提示没有出现

提示只用于 `marker`，且 `labelFields` 必须使用 DataFrame 中准确的字段名。所有指定
字段在当前行都为空时不会显示提示。

### 更换凭据后提示 SDK 已使用其他凭据加载

重新加载整个 Grafana 页面。高德 SDK 在页面内单例加载，同一页面不支持多组凭据。
